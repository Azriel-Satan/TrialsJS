Tits JS Command List

Welcome to the Tits JS Command List

What we strive to do here is create a list containing a wide variety of commands that can be used to safely augment the playing experience outside of what the developers intended by using the Web Console to interface with the Javascript the game is built from.
If something you use here breaks your game, blame yourself, then Me, but not the Devs; they never intended for the game to be played in this way.

Setup

The first thing you will need to do for the command list to be fully operational is input the location of your image pack. If you are playing online use the URL of the Playable build on the Fenoxo Website, if playing locally use the location of the game install.

https://www.fenoxo.com/play/TiTS/release
[path to exe folder]\resources\app
[path to ImagePack]\resources\app

Thanks to inconsistencies in how the imagepack is generated, some images may be missing on the web source.
The local version requires the x2 and x4 for some characters.

It will save your input, if you want to change the path submit the new text and refresh.
